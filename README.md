## Intro

使用vue.js結合firebase即時資料庫建立線上聊天室

### 聊天功能

需先輸入名字才能夠開始聊天，聊天時，自己的名字會以灰色顯示，其他人的名字會以深藍色顯示

一旦開始聊天後，名字會預設為不可修改，若想要修改，可點選旁邊的"X"按鈕以輸入新名字


### 暫存使用者名稱

開始聊天後，會自動儲存使用者所輸入的名字，下次造訪聊天室時，可看到自己上次聊天所用的名字

若要修改或清除紀錄，可點選名字輸入框旁邊的"X"按鈕
